MapBox Locator
==============

A widget builder for creating simple locator maps placing one or more markers on a [MapBox](http://www.mapbox.com) map.

![Screenshot](http://i.imgur.com/CkZToYl.jpg)

## Starting Locator

To run locally, run:

> python -mSimpleHTTPServer

and go to [http://localhost:8000](http://localhost:8000)

---

Requires a MapBox Map ID in the initialize function

```javascript
// Insert your Map ID here
var mapId = 'examples.map-vyofok3q';
```
